## 已实现的 4 个需求

### 1. ✅ 欢迎消息（Bot 启动时发送）
- `BotCommandService.onBotConnected()`
- 获取用户信息，发送个性化欢迎消息
- 包含可用命令列表和配置状态

### 2. ✅ 链接消息处理（https://t.me/CRAVWM/38700）
- `TelegramClientService.handleUpdate()` 中检测消息中的 t.me 链接
- 通过 `LinkMessageEvent` 分发
- `BotCommandService.handleLinkMessage()` 接收并下载

### 3. ✅ 自动从 Saved Messages 下载
- 连接成功后自动检查配置的聊天
- Saved Messages = `chatId = -userId`
- 默认添加 Saved Messages 到下载列表
- 定时检查新消息（每 30 秒）
- 记录最后读取消息 ID

### 4. ✅ 不重复下载
- `DownloadRecord` 实体记录已下载文件
- 基于 `fileUniqueId`（TDLib 文件 ID）判断
- 内存缓存 `downloadedFileSet` 加速查询

## 架构

```
TelegramClientService (事件生产者)
    ↓ TelegramEventDispatcher (事件分发)
    ↓
BotCommandService (事件消费者)
    ├── ConnectedEvent    → 发送欢迎消息 + 初始化
    ├── DisconnectedEvent → 更新状态
    ├── CommandEvent      → 处理命令 (/start, /help, /download, ...)
    └── LinkMessageEvent  → 处理链接消息
```

## 关键代码路径

1. **Telegram 连接** → `TelegramClientService.connect()`
2. **事件分发** → `TelegramClientService.handleUpdate()`
3. **Bot 响应** → `BotCommandService.onTelegramEvent()`
4. **自动下载** → `BotCommandService.checkAndAutoDownload()`
5. **去重检查** → `BotCommandService.downloadMediaFromMessage()`
